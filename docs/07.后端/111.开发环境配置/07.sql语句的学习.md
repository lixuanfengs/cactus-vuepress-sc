### 简单实践SQL

#### DDL创建数据库

DDL:创建数据库,删除数据库,添加表字段，表索引

```yaml
# 创建数据库lol
# 数据库里大小写不敏感(不是不区分大小写)
# 内置的sql关键字全部大写,然后自定义的数据,可以进行大小写管理 
mysql> CREATE DATABASE lol;
Query OK, 1 row affected (0.00 sec)

mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| lol                |
# 删除数据库
mysql> DROP database lol;
#最完善的写法,创建数据库,数据库存在也不会报错
mysql> CREATE DATABASE IF NOT EXISTS log;
Query OK, 1 row affected (0.00 sec)

mysql> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| log                |
| lol                |
# 创建数据库并设置字符集utf-8
mysql> CREATE DATABASE IF NOT EXISTS los DEFAULT CHARSET UTF8 COLLATE utf8_general_ci;
# 查看数据库创建的sql是什么
mysql> SHOW CREATE DATABASE los;
+----------+--------------------------------------------------------------+
| Database | Create Database                                              |
+----------+--------------------------------------------------------------+
| los      | CREATE DATABASE `los` /*!40100 DEFAULT CHARACTER SET utf8 */ |
+----------+--------------------------------------------------------------+
1 row in set (0.00 sec)
```

##### 授权特性

```yaml
1、权限:不要使用all privileges，尽量指定某一个,某几个权限即可
2、主机范围,尽量别用%,尽量别给一个局域网的网段或者限制某一个ip,即使是需要在公网环境连接,使用%号也要通过防火墙做一定的安全措施以及更换MySQL默认端口等
3、grant授权对数据库的选择,也尽量别用*,而指定数据库以及数据表
1、root登录密码复杂
2、更改ssh端口
3、禁止root登录,只允许普通用户登录
4、禁止用账号密码登录
5、只用普通用户+公私钥的形式登录
6、修改常见软件的默认端口,如3306 6379等等
```

#### MySQL数据备份

备份就是MySQL的各种数据库(database)以及库内的各种数据表(table)

备份的方案有多种

1、只备份每一个库,不要里面的表

2、备份所有的库和所有的表,但是不要表中的数据,只备份它的结构(创建的sql)

3、只要某一个库以及这个库中的所有表以及所有数据

4、只要某一个库里面的某一张表

MYUSER=root
MYPWD=xiaomao.net
DBPATH=/mysql_db_back/
MYCMD="mysql -u$MYUSER -p$MYPWD"
MYDUMP="mysqldump -u$MYUSER -p$MYPWD"
[ ! -d "$DBPATH" ] && mkdir $DBPATH
for dbname in `$MYCMD -e "show databases;"|sed '1d'|egrep -v "mysql|schema"`
do
        # 创建数据库同名文件夹
        mkdir ${DBPATH}/${dbname}_$(date +%F) -p
        # 循环找出所有数据表
        for table in `$MYCMD -e "show tables from $dbname;"|sed '1d'`
        do
          $MYDUMP $dbname $table  > $DBPATH/${dbname}_$(date +%F)/${dbname}_${table}.sql

#### 总结

```python
# 选择查询
select *  from   表名   where   条件
# 插入
insert  into 表名(字段1,字段2)  values (值1,值2)
# 删除
delete from 表名  where 范围
# 更新
update 表名  set 字段1=值1  where   条件
#  查询所有
select *  from 表名 
#  查找
select  *  from 表名   where 字段1  like  '%值1%'
#  排序
select *  from  表名  order by   字段   desc
# 总数 
select  count  as  字段   from  表名
#  联表查询
select  * from   左表名  left join  右表名   on  条件  union  左表名  right  join 右表名  on  条件 
```
